# NBAPLAY

NBAPLAY is a comprehensive project designed to scrape match statistics for over 200 players, manipulate and present the data dynamically. The project is divided into three main components: Backend, Frontend, Data Scraping.

You can find the [project here]()

## Features
- **Data Scraping**: Engineered a data scraping of match statistics for 200+ players using Python and pandas
- **Backend**: Dynamic manipulation and presentation of the scraped data through a Spring Boot application
- **Database**: Real-time data manipulation within a Postgres database using SQL queries
- **Frontend**: Seamless integration with a user-friendly ReactJS interface
